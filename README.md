# SYDE362-Wifi-Scanner

Automated RSSI Wi-Fi Scanner For an Android N+ Device.
Built for SYDE 362 Assignment P3.

### Background

Leverages Android WifiManager API to periodically scan and log the RSSI values of your device's Wi-Fi connection.
Collects data in 5 second intervals over 30 data points, totalling 150 seconds.
